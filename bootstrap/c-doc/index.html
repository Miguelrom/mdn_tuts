Introduction

The C programming language was designed by Denis Ritchie in 1972 at Bell Labs to develop utilities for the UNIX operating system. Today, C is one of the most popular programming languages despite being almost 60 years old.

The C language is mostly used to develop software that works closely to the hardware like operating systems, device controllers, compilers, etc. Although, you can also find desktop applications written in C. Some of the features that make the C language special are the following:

  • Easy to learn.
  • It produces fast running programs.
  • It can be compiled in almost every computer platform.
  • Comprehensive standard and third party libraries.
  • Widely used in open source software.

Hello World

To create a program that prints "Hello World!" to the console, create a file called hello.c and paste in it the following code: 


  #include <stdio.h>

  int main() {

      printf("Hello World!\n");

      return 0;
  }

Open a terminal and move to the directory where you saved the file. Now, run the following command: 

  gcc hello.c -o hello

To run the executable just created, type in:

  ./hello

You should see that the terminal outputs this message:

  Hello World!

Congratulations!!! You just compiled and run your first C program.

Primitive Data Types

There are only four basic data types in C:

  • char: data type to store a character.
  • int: data type to store an integer number.
  • float: data type to store a floating point (real) number.
  • double: data type to store a double-precision float.

Variables

Variable names are made up by letters and numbers; the fisrt character must be a letter. The underscore "_" counts as a letter. Don't begin a variable with an underscore, though, since library routines often use such names. Also, variables are case sensitive; this means that "x" is different from "X".

All variables must be declared before they're used. A declaration specifies a type, and contains a list of one or more variables of that type. Example: 


  int lower, upper, step;
  char c, line[1000];
  float PI=3.1416;
  double circleArea;

Remember to give meaningful and related to their purpose names to your variables. It is also a good idea to use cames case format for variables with multiple names. For example: myVariable, randomNumber, userName, etc.

Scope of a Variable

The scope of a variable means the places within the source code a variable can be called or referenced. Usually the scope of a variable is enclosed in curly braces ({}). A code block inside curly braces can be a function, loop, struct, etc. You cannot call a variable outside the curly braces block it was declared in. But you can call a variable inside any nested curly braces block.

if Statement

The if statement determines the execution of a piece of code based on the validity of a condition. Example: 


  if (number > 3) {
    printf("The number is greater that 3/n");
  }

You can also evaluate two or more cases using the if-else statement:


  if (number > 3) {
    printf("The number is greater that 3/n");
  }else if (number < 3){
    printf("The number is less that 3/n");
  }else {
    printf("The number is unknown\n");
  }

switch Statement

In situations where you need to evaluate multiple conditions you could use the swtich statement which has the following syntax: 


  switch (expression){
    case const-expr: statements;
    case const-expr: statements;
    default: statements;
  }

The expression is usually a variable and const-expr a possible value for that variable. When the expression matches a case, execution starts right there. The case labeled default is optional and is executed if none of the other cases are satisfied. Here's an example:


  int number = 5;

  switch (number) {
    case 1: printf("The number is 1\n");
            break;
    case 2: printf("The number is 2\n");
            break;
    case 3: printf("The number is 3\n");
            break;
    case 4: printf("The number is 4\n");
            break;
    case 5: printf("The number is 5\n");
            break;
    default: printf("The number is unknown.\n");
  }

The break statement causes an immediate exit from the switch. After the code for one case is done, execution falls through the next unless you take explicit action to escape.

for Loop

The for loop statement has the following general form:


  for (expr1; expr2; expr3){
    statements
  }

The order of execution is as follows:

  1. expr1.
  2. expr2.
  3. statements.
  4. expr3.
  5. Back to step 2.

Most commonly expr1 is a variable that serves as a counter, expr2 is a relational statement and expr3 increments or decrements the value of the counter variable. Example: 


  for (int i = 1; i >= 5; i++){
    printf("The number is %i", i);
  }
The statements inside the for loop block are executed until the relational expression evaluates to zero.

while Loop

The while loop is a simpler version of the for loop. The statements inside the while block are executed so long the expression is not equal to zero. Here's its general form: 


  while (expression) {
    statements
  }

We can rewrite the example of the for loop using the while loop like so: 


  int i = 1;

  while (i <= 5){
    printf("The number is %i", i);
    i++;
  }

Note: you've got to make sure you include a way to eventually stop the loop, otherwise the statements inside the while block will run forever.

do while Loop

The do while loop executes its block of statements one time before it evaluates its conditional expression. Its general form is the following: 


  do{
    statements
  }while(expression);

You would want to use this loop every time you need to execute a set of statements at least one time. For example: 


  int a = 1;

  do{
    printf("%i is less than 5.\n", a);
    ++a;
  }while(a < 5);

Arrays

An array is a data structure that stores data items in a linear fashion so you can access them by a numerical index. The way you declare an array is as follows: 


  data_type array_name[size];

Arrays store data items of the same type. The size of an array is the number of items it contains, which you have to specify in the declaration. You acces an item by passing its position (called index) within the array. The indexing starts at zero and ends at n-1. Let's see an example: 


  int myIntegers[5];

  myIntegers[0] = 7;
  myIntegers[1] = 2;
  myIntegers[2] = 4;
  myIntegers[3] = 19;
  myIntegers[4] = 12;

  for (int i = 0; i < 5; ++i){
    printf(myIntigers[i]);
  }

Functions

Functions are a way to pack common operations for the purpose of organization and reutilization. The general form of a function is as follows: 


  retuned_type function_name (parameters){
    statements
    return expression;
  }

Let's suppose we want a function that raises a number to a certain power. That function would be something like this: 


  #include <stdio.h>

  int pow (int base, int power){
    int result = base;
    for (int i = 2; i <= power; ++i){
      result *= base;
    }

    return result;
  }

  int main {
    int result = pow (2, 4);
    printf("2^4 = %i", result);
  }

Pointers

Pointers are a special data type for variables that point to memory addresses of other variables. The way you declare a pointer variables is like so: 


  data_type * variable_name;

The unary operation & gives the address of an object, so the statement p = &c; assigns the address of c to the varialbe p, and p is said to "point to" c.
The unary operator * is the indirection or dereferencing operator; when applied to a pointer, it access the object the pointer points to. Let's see an example: 


  int x = 5;
  int * ptr;
  ptr = &x;
  /*This prints: "ptr = 5"*/
  printf("ptr = %i", *ptr);

Structures

A structure is a collection of one or more variables, possibly of different types, grouped together under a single name for convenient handling. Structures help to organize complicated data, particularly in large programs, because they permit a group of related variables to be treated as a unit instead of as separate entities.
A good example for a structure is a point, which consist of a x coordinate and a y coordinate, both integers: 


  struct point {
    int x;
    int y;
  };

The keyword struct introduces a structure delcaration, which is a list of declarations enclosed in braces. An optional name called a structure tag may follow the word struct (as with point here). The tag names this kind of structure, and can be used subsequently as a shorthand for the part of the declaration in braces.
Given the declaration of point above, the following defines a variable pt which is a structure of type struct point: 

  struct point pt

The variables named in a structure are called members. A member of a particular structure is referred to in an expression by a contruction of the form: 

  structure-name.member

The structure member operator "." connects the structure name and the member name. To print the coordinates of the point pt, we could write: 

  printf("x = %d, y = %d", pt.x, pt.y);
